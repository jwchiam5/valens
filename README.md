# Valentine's Day Website

This project is a fun interactive Valentine's Day website built with Vite and React. The website prompts the user "Will you be my Valentine?" and features interactive buttons that resize based on user interaction.

## Features
- Interactive buttons that resize based on user interaction
- Responsive design with Valentine's theme
- Auto-refresh development server with Vite

## Development
- **Framework**: React with Vite
- **Styling**: CSS with Valentine's theme colors
- **Development server**: Vite dev server with hot reload

## Getting Started

To get started with the project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd valentines-website
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the development server**:
   ```
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:3000` to see the application in action.

## Usage

The application will prompt you with "Will you be my Valentine?" and you can interact with the buttons. Clicking "No" will make the "No" button smaller and the "Yes" button bigger until you eventually click "Yes".
